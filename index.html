🔬 Applied AI Research · Western University · 2025–2026
CODEBUSTER — Hybrid Similarity Model for C++ Plagiarism Detection

A multilayered plagiarism detection system combining TF-IDF cosine similarity, a fine-tuned GraphCodeBERT model (with function-level AST decomposition and data flow graphs), and sandboxed output-based behavioral comparison. The three signals are fused into a feature vector fed through an MLP classifier, achieving 79.8% accuracy on 3,159 held-out C++ code pairs — outperforming token-only tools like MOSS and JPlag against AI-assisted obfuscation.

with Aaron Yi · Arjun Krishnan · Munadir Khan · Riley Wong · Khilan Desai
